# 03 — CLT Company Hero Section

## Idea
Exploration for a CLT (Cross-Laminated Timber) company that builds in wood. A hero section where the wooden skeleton/structure is masked with the finished building image, revealing the construction beneath.

## Core Concept
- Two layers: the raw wooden structure and the finished building
- Use CSS or canvas masking to reveal the skeleton underneath
- Could be scroll-driven, hover-based, or use a slider to transition between the two states
- Tells the story of what's hidden inside CLT buildings

## Tech to Explore
- CSS `mask-image` / `clip-path`
- Scroll-driven animations
- Canvas compositing
- Before/after slider patterns
